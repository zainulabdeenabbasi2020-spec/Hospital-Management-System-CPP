# Hospital Management System (OOP) 🏥

A C++ application designed to manage hospital operations like patient records and doctor details using Object-Oriented Programming (OOP) principles.

## 🚀 Key Features
- **Patient Management:** Store and display patient details.
- **Doctor Records:** Maintain doctor specializations.
- **Billing Module:** Basic logic for managing hospital bills.

## 💻 OOP Concepts Used
- **Inheritance:** Used a base `Person` class for `Patient` and `Doctor`.
- **Polymorphism:** Implemented virtual functions for dynamic method overriding.
- **Encapsulation:** Used access specifiers (private/protected) to secure data.
- **Abstraction:** Defined pure virtual functions in the base class.

## 🛠 Tech Stack
- **Language:** C++
- **Environment:** Any standard C++ Compiler (GCC/Clang)
